# How to compile.
1. Clone the repo.
1. Open in Unity.
1. Wait until it reimports everything.
1. Hit play.
1. It probably doesn't work.
1. If so, delete **Plugins** folder.
1. Open **Services** window (ctrl+0).
1. IAP
1. Import -> Install Now -> Import everything
1. It should work now.
